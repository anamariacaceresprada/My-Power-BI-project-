# Power BI Retail & Sales Data Analysis Project

## Overview
This project was completed as part of a **Data Technician Bootcamp** and focuses on analysing retail and sales data using **Microsoft Power BI**. The goal of the project was to transform raw data into meaningful insights through **data cleaning, transformation, and interactive visualisations**.

The project demonstrates key business intelligence skills including **data preparation, data modelling, and dashboard design**, allowing users to explore trends and performance metrics through interactive reports.

---

## Key Skills Demonstrated

### Data Transformation & Cleaning (Power Query)
- Imported and prepared raw datasets using **Power Query**
- Cleaned and transformed data by:
  - Removing errors and duplicates
  - Renaming and formatting columns
  - Changing data types
  - Structuring data for analysis
- Prepared the dataset for efficient reporting and analysis

---

### Data Modelling & DAX
Used **DAX (Data Analysis Expressions)** to enhance the dataset and support deeper analysis:

- Created **calculated columns** to derive new data attributes
- Built **measures** to calculate key business metrics such as totals and averages
- Structured calculations to support dynamic reporting and filtering

---

### Interactive Reports
Developed interactive reports that allow users to explore data dynamically:

- **Slicers** to filter reports by category, region, or time period
- **Filters** to refine analysis and focus on specific data segments
- Dashboard layouts designed for clear navigation and exploration

---

### Data Visualisation Techniques
Created a variety of visualisations to communicate insights effectively:

- **Bar charts** to compare sales performance across categories
- **Line charts** to display sales trends over time
- **Pie charts** to show proportional distribution of sales
- **Map visualisations** to analyse geographic sales patterns

These visualisations help convert raw data into **clear and actionable insights**.

---

## Data Storytelling
A key objective of this project was **data storytelling**—using visualisation to communicate insights clearly.

The reports were designed to:
- Highlight key retail performance indicators
- Reveal trends and patterns in sales data
- Enable users to interact with the data to uncover insights

---
